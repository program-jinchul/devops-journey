## Terraform variables.tf, outputs.tf 실습

### 배운 것

**variables.tf**
- 변수 파일로 하드코딩 제거
- variable 블록으로 변수 선언
- description: 변수 설명, default: 기본값

**main.tf 변수 참조**
- `var.변수명` 으로 참조
- 따옴표 안에 넣으면 문자열로 인식되어 에러 발생
- `"var.instance_type"` (❌) → `var.instance_type` (✅)

**outputs.tf**
- 생성된 리소스 정보 출력
- output 블록으로 출력값 선언
- terraform apply 후 하단에 자동 출력

**Terraform 파일 구조**
- main.tf: 리소스 정의
- variables.tf: 변수 정의
- outputs.tf: 출력값 정의
- .terraform/: provider 플러그인 (gitignore 처리)

### 트러블슈팅
- 문제: 변수 따옴표 안에 넣어서 에러 발생
- 해결: 따옴표 제거 후 정상 작동
