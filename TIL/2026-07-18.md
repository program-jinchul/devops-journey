## boto3 통합 자동화 스크립트

### 배운 것

**S3 파일 삭제**
- `s3.delete_object(Bucket=버킷명, Key=파일명)`
- Key는 S3에 저장된 파일명 기준 (로컬 경로 아님)

**AWS Manager 클래스 설계**
- EC2, RDS, S3 클라이언트를 하나의 클래스로 통합
- `__init__` 안에서 boto3 클라이언트 생성하는 방식이 외부에서 주입하는 것보다 깔끔
- 각 서비스별 메서드로 분리 (ec2_monitor, rds_monitor, s3_monitor)

**통합 실행 흐름**
- 사용자 입력으로 서비스 선택 (ec2/rds/s3)
- 서비스에 따라 action 분기 (start/stop/upload/delete)

### 느낀 것
- 클래스로 묶으니까 코드가 훨씬 깔끔하고 확장하기 쉬워짐
- 지금까지 짠 boto3 코드들이 하나로 합쳐지는 느낌
