## Nginx HTML 페이지 커스터마이징

- 기존 Nginx 기본 페이지를 커스텀 HTML로 교체
- nano 에디터로 서버 파일 직접 수정
- UTF-8 인코딩 설정으로 한글 깨짐 해결
- 브라우저에서 실제 서비스 확인 완료

## RDS 생성 및 EC2 연결
- DB Subnet Group 생성
- RDS MYSQL 인스턴스 생성 (free tier, jinchul-vpc)
- Security Group 3306 포트 인바운드 규칙 추가
- EC2애서 MYSQL 클라이언트로 RDS 접속 성공
- jinchuldb 데이터베이스 생성 확인

## 트러블슈팅
- RDS 접속 실패 원인: jinchul-db-sg에 3306 포트 규칙 없음
- 해결: EC2 Security Group 소스로 3306 인바운드 규칙 추가
