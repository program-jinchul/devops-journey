## SQL 심화 실습 (LIKE, LEFT JOIN, 서브쿼리)

### 배운 것

**LIKE** — 문자열 검색
- `'j%'` : j로 시작하는 것
- `'%ng'` : ng로 끝나는 것
- `'%in%'` : in이 포함된 것
- `%` 는 와일드카드

**JOIN vs LEFT JOIN**
- JOIN : 양쪽 테이블에 모두 있는 데이터만 조회
- LEFT JOIN : 왼쪽 테이블 기준 전부 조회, 오른쪽에 없으면 NULL

**IS NULL** — NULL 값 찾기
- `WHERE orders.id IS NULL`
- `= NULL` 이 아니라 `IS NULL` 써야 함

**서브쿼리** — 쿼리 안에 쿼리
- 안쪽 쿼리 먼저 실행 후 바깥 쿼리에 결과 전달

**LIMIT** — 개수 제한
- `LIMIT 1` : 결과 1개만 가져오기

**AVG** — 평균
- `AVG(price)` : 가격 평균

### 문제 풀이
- 가장 비싼 상품 주문한 유저 이름 조회 (서브쿼리)
- 평균 가격보다 비싼 상품 조회 (AVG + 서브쿼리)
- 주문 안 한 유저 이름/이메일 조회 (LEFT JOIN + IS NULL)
