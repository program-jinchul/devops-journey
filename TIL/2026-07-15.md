## boto3 S3 실습

### 배운 것

**S3 버킷 생성**
- AWS 콘솔에서 jinchul-devops-bucket 생성
- Block all public access 설정으로 보안 유지

**파일 업로드**
- `s3.upload_file(파일경로, 버킷명, 저장될파일명)`
- 로컬 파일을 S3 버킷에 업로드

**파일 목록 조회**
- `s3.list_objects_v2(Bucket=버킷명)`
- `Contents` 키로 파일 목록 접근
- Key(파일명), Size(크기) 출력

### 느낀 것
- EC2, RDS와 마찬가지로 boto3 구조는 딕셔너리 기반으로 동일함
- S3는 EC2/RDS보다 간단하게 제어 가능
