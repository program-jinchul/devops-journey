## IAM Role 생성

### 배운 것

**IAM 핵심 3가지**
- User - AWS에 로그인하는 사람
- Role - AWS 서비스가 다른 서비스에 접근할 때 쓰는 권한
- Policy - 실제 권한 내용

**IAM 생성 실습**
- EC2에 IAM Role를 붙여서 EC2가 S3에 접근할 수 있게 붙임
- Policy 연결 (AmazonS3FullAccess) 이게 S3 접근 권한
- EC2에 Role 붙이기 완료!

### 느낀 것
- EC2에 IAM Role를 붙여서 EC2 서버 안에서 boto3키 없이도 S3에 접근가능
- EC2가 Role에서 자동으로 권한을 가져옴
