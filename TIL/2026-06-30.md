## SQL 심화 실습 (ORDER BY, COUNT, HAVING)

### 배운 것

**ORDER BY** — 정렬
- DESC : 내림차순
- ASC : 오름차순

**COUNT** — 개수 세기
- COUNT(컬럼명) 으로 행 개수 집계

**HAVING** — 집계 결과에 조건 걸기
- GROUP BY 이후에 사용
- COUNT, SUM 같은 집계함수에 조건 걸 때 사용

### WHERE vs HAVING
- WHERE : 그룹화 전, 행 단위 필터
- HAVING : 그룹화 후, 집계 결과 필터

### 문제 풀이
- users 이름순 정렬
- orders 5만원 이상 필터
- 유저별 총 구매금액 10만원 이상 필터
