## Terraform 설치 및 명령어 실습

### 배운 것

**Terraform 설치**
- terraform install windows 검색 후 windows amd64버전 설치
- zip파일 다운로드 후 압축해제
- C파일에 terraform 파일 만듬 거기에 zip파일 terraform.exe 옮김
- 시스템 횐경 변수 편집에서 C파일에 추가한 terraform 환경변수 PATH에 추가
- 터미널에 terraform --version으로 확인

**Terraform 초기 설정**
- devops-journey 폴더에 새로 만든 terraform/ 폴더에 main.tf 파일 생성
- main.tf에 required_providers로 AWS provider 선언
- provider "aws"로 서울 리전 설정
- .gitignore에 .terraform/, terrform.tfstate 추가(민감 파일 제외)

**Terraform 명령어 실습**
- terraform init: provider 플러그인 설치 및 초기화 
- terraform plan: 배포 전 어떤 리소스가 만들어질지 미리 확인
- terraform apply: 실제 AWS에 리소스 배포
- terraform destroy: 배포된 리소스 삭제

**자동 생성 파일**
- .terraform/: provider 플러그인 설치 폴더
- terraform.tfstate: 현재 인프라 상태 저장 파일
- terraform.tfstate.backup: 이전 상태 백업 파일
