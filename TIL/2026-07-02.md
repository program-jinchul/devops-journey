## SQL 마무리 + README 정리

### SQL 추가 학습

**UNION** — 두 쿼리 결과를 아래로 합치기
- JOIN은 컬럼 기준으로 옆으로 합치는 것
- UNION은 행 기준으로 아래로 합치는 것

**CASE** — 조건에 따라 다른 값 출력
- Python의 if/elif/else와 같은 구조
- WHEN = elif, ELSE = else, END로 블록 닫기
