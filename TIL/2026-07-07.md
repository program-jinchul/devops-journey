## boto3 실습 - EC2/RDS 자동화

### 배운 것

**boto3 환경 설정**
- `aws configure` 로 IAM Access Key 설정
- boto3가 `~/.aws/credentials` 에서 자동으로 키 읽어옴

**EC2 자동화**
- `describe_instances()` : EC2 목록 및 상태 조회
- `start_instances()` : EC2 시작
- `stop_instances()` : EC2 중지
- input() 으로 사용자가 start/stop 선택하게 구현

**RDS 자동화**
- `describe_db_instances()` : RDS 목록 및 상태 조회
- `start_db_instance()` : RDS 시작
- `stop_db_instance()` : RDS 중지
- EC2와 달리 InstanceIds 리스트가 아닌 문자열로 ID 전달

### 트러블슈팅
- RDS stop 시 InvalidDBInstanceStateFault 에러
- 원인: starting 상태에서 stop 불가
- 해결: available 상태 된 후 실행
