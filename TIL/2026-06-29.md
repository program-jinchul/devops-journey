## SQLD 기초 실습 (RDS MYSQL) - 1

### 배운 것

**CRUD**
- INSERT : 데이터 삽입
- SELECT : 데이터 조회
- UPDATE : 데이터 수정 (WHERE 없으면 전체 수정)
- DELETE : 데이터 삭제 (WHERE 없으면 전체 삭제 주의)

**WHERE**
- 조건, 필터, 원하는 데이터만 골라서 조회

**JOIN**
- 두 테이블을 공통 컬럼 기준으로 합쳐서 조회
- ON 뒤에 기준 컬럼 지정 필수

**SUM / AS**
- SUM() : 합계
- AS : 컬럼에 별칭 붙이기

**GROUP BY**
- 특정 컬럼 기준으로 묶어서 집계
- 유저별, 날짜별 통계 낼 때 사용
